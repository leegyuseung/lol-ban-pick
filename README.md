<h2>MetaPick ☑️</h2>

사용자들은 일반 밴픽 모드뿐만 아니라 피어리스 밴픽 모드까지 활용하여 다양한 전략을 연습하고 실험할 수 있습니다. <br />

직관적인 인터페이스와 실시간 상호작용 기능을 통해 보다 체계적인 밴픽 연습이 가능하도록 설계되었습니다. <br />
